## Description
This project is a simple chatbot created using Python. 
The chatbot responds to predefined user inputs using rule-based conditions.

## Features
- Responds to greetings
- Answers simple questions
- Uses if-elif conditions
- Easy to modify

## Requirements
- Python 3.x

## How to Run

1. Save the code as chatbot.py
2. Open terminal
3. Run:

python chatbot.py

## Example

You: hello
Bot: Hi! How can I help you?

You: what is your name
Bot: I am a Rule-Based Chatbot.

You: bye
Bot: Goodbye!
